🚀 HOW TO USE

Step 1: Database Setup
   psql -U postgres -c "CREATE DATABASE fn_furniture_inventory;"
   psql -U postgres -d fn_furniture_inventory -f database/schema.sql

Step 2: Backend
   cd backend
   npm install
   cp .env.example .env
   # Edit .env with your database password
   npm run dev

Step 3: Frontend
   cd frontend
   npm install
   cp .env.example .env
   npm start

Step 4: Development
   Edit controllers, routes, services, components
   All changes will hot-reload
   Commit with meaningful messages
   Create feature branches from initial-structure
